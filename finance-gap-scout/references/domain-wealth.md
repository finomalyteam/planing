# Domain: Wealth — Wealth Management, Private Banking, Robo-advisor

## ขอบเขตสายงาน

- บริการบริหารความมั่งคั่ง: financial planning, portfolio advisory, private banking
- Robo-advisor และ hybrid advisory (คน + ระบบ)
- เครื่องมือของ advisor (RM tools): CRM, proposal, rebalancing, reporting
- Segment: HNW, affluent, mass affluent, first jobber

## ผู้เล่นที่ควรสำรวจ

- ไทย: Finnomena, StashAway (ถอนตัว/สถานะปัจจุบัน — เช็คให้ชัด), odini,
  หน่วยงาน wealth ของธนาคารใหญ่ (SCB CIO/Julius Baer JV, KBank Private Banking +
  Lombard Odier, Krungsri Exclusive ฯลฯ)
- Global robo: Betterment, Wealthfront, Nutmeg, Endowus, Syfe
- Advisor tech: Addepar, Orion, eMoney, Conquest Planning, intelliflo

## จุดที่มักเป็น gap ในสายนี้ (hypothesis ให้ไป verify)

- บริการวางแผนการเงินแบบองค์รวม (หนี้+ประกัน+ภาษี+ลงทุน) สำหรับ mass affluent
  ยังแพงหรือไม่มี — robo ส่วนใหญ่ทำแค่จัดพอร์ตกองทุน
- เครื่องมือ RM ไทยยัง manual สูง: เตรียม proposal, สรุปพอร์ตลูกค้า, บันทึกหลังพบลูกค้า
- การสื่อสารตอนตลาดผันผวน (proactive reassurance) ยังเป็นโทรมือถือ/LINE ไล่ทีละคน
- Decumulation (วางแผนถอนเงินหลังเกษียณ) ถูกพูดถึงน้อยกว่า accumulation มาก

## ตัวเลขที่ควรหาให้เจอ (สำหรับ cost story)

- ค่าธรรมเนียม advisory เฉลี่ย (% ของ AUM) เทียบ robo
- Cost-to-income ratio ของธุรกิจ wealth
- จำนวนลูกค้าต่อ RM หนึ่งคน และสัดส่วนเวลาที่หมดกับงาน admin
- ขนาดตลาด mass affluent ไทย (จำนวนคน/สินทรัพย์)
